# 📚 StudySphere – AI-Powered Study Tracker

StudySphere is a futuristic study productivity tracker designed for students to manage study sessions, monitor progress, track goals, and visualize analytics through an immersive dashboard.

---

## 🚀 Features

- 📖 Study Session Tracking
- ⏱️ Pomodoro Timer
- 📊 Analytics Dashboard
- 🎯 Goal Management
- 🔥 Study Streak Tracking
- 📝 Smart Notes & Revision
- 📁 CSV Export
- 🔐 Secure Authentication
- 🌙 Glassmorphism UI

---

## 🛠️ Tech Stack

### Backend
- Flask
- SQLite
- Flask-Login
- Flask-Bcrypt
- SQLAlchemy

### Frontend
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Chart.js
